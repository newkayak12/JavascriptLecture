                변수는 접근할 수 있는 범위에 따라 영역을(scope) 구분한다.

                영역은  global scope/ local scope로 나눈다.
                global scope : 페이지 내부 어디서든 접근이 가능한 변수 > window 객체의 멤버 변수가 된다.(window는 페이지 전체를 포함함)
                local scope : function 내부에서 선언한 변수 > function 내부에서만 사용이 가능하다.

자바스크립트 자료형

                타입을 지정하지 않는다고해서 자료형이 없는게 아니에요
                변수에 리터럴 값이 주입 될 때 타입이 결정된다.

                자바스크립트는 자료형을 가지고 있으며, 변수의 자료형이 결정되는 것은 
                변수에 값(리터럴) 대입할 때이다.

                각 자료형을 알아보고, 자료형 리터럴 표현 방법을 알아보자
                문자열(String), 숫자(number), 배열([]), 객체({}), 함수(function), undefine, Boolean

변수의 타입 확인하기

                typeof라는 함수를 이용하면 타입을 알 수 있다.
