자바스크립트 이용하기

             자바스크립트로 페이지를 interactive하게 구성하려면
             자바스크립트 구문을 태그 내 속성(inline), script태그 내부, 외부 js파일에 작성한다.
  1. inline : 태그 내부의 이벤트 속성(on _ _ _)에 직접 javascript코드를 작성하는 것 * 주로 간단한 구문
  2. internal : 페이지의 script 태그 안에 javascript코드를 작성하는 것
  3. external : 따로 js파일을 만들어서 html안으로 javascript코드를 작성하고 페이지에서 script태그를 이용해서 불러와(script scr="주소" 사용 /서버 내부 저장된 파일이나 외부에서 지원하느 파일 두 개 다 가능
