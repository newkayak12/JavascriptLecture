window 객체

                창 하나마다 window객체가 생긴다. _ 탭도 하나의 창이니까... 당연히 window객체가 생기겠지


                윈도우 객체는 자바 스크립트의 최상위 객체로 모든 객체와 함수를 가지고 있으며, 기본객체로 설정되어있다. 
                brower한 개마다 한 개의 윈도우 객체가 생성된다. 

                윈도우 객체의 주요 자식 객체
                    Browser Object Model : 
                        location  : 브라우저 url주소를 설정 및 정보
                        navigator : 브라우저에 대한 정보, 미디어스트리밍 등의 설정을 가져옴
                        screen    : 사용자 화면에 대한 정보
                        history   : 페이지 이동 이력 정보를 가져온다. 
                
                    Documnet Object Model : 
                        document  : 브라우저의 HTML 태그들을 객체화해서 처리하는 객체
                    

                    주요 함수 : 윈도우를 조작하고, 다른 추가 기능(입력, 경고(알림), 새 창 열기, 주기적인 실행 등)
                    ex_ resizeTo() :
                        moveTo() : 
                        focus() : 
                        height
                        width 
                        alert
                        prompt
                        confirm
                        setInterval
                        setTimeout

윈도우 함수 이용하기

open()

                새 창을 생성하는 함수 
                window.open(" 연결할 URL "," target 속성 "," 새로 여는 창에 대한 설정 ");(window 생략 가능)
                    연결할 URL : 새창을 열 페이지 주소 _ http://~~~.com > 공란은 blank
                    target 속성 : 새 창에서 열 것인지 아니면 현재 창에서 옮길지 (_self, _blank)
                    새로 여는 창에 대한 설정 : 크기 _ 브라우저를 탐

                    open()함수는 새로 생긴 window객체를 반환한다.
