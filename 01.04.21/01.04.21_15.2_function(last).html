ES6화살표 함수 알아보기

        자바에서 람다 처럼 ()->{}
        익명함수를 표현할 때 가독성이 좋게 표현을 한 것을 화살표 함수라고 함
        익명함수 : function() {  };
        화살표 함수 : () => { };
        
        익명함수 : function(su1, su2){  return };
        화살표 함수 : (su1, su2) -> { return  };
