                    location 객체 : url에 뭔가 적으면 : 요청하는 것 _ / url에 대한 정보를 가져 올 수도 있다.

                    navigator : 브라우저에 대한 정보 / 브라우저 코드명, 버젼, OS에 대한 정보까지

screen

                    window객체의 property객체로 client화면에 대한 정보를 가지고 있는 객체

navigator

                    window 객체의 property객체로 client의 browser 정보를 가지고 있는 객체

location

                    window객체로 property객체로 브라우저의 url정보를 가지고 있는 객체
                    url정보를 변경하여 페이지를 이동할 수 있다. 
                     
                    * 주로 페이지 변경할 때 많이 쓴다. 
                        1. 함수를 이용
                            assign() : history가 남는다. (뒤로가기를 허용하는 곳)
                            replace() : history가 안 남는다. (뒤로가기를 허용하면 안되는 지점에서 사용)
                            reload()

                        2. property를 변경
                            location.href="주소"; : url 주소가 변경 됨 (다시, url에 값을 적는다 = 요청 무조건 응답이 있다. (htmlfile))
                            (assign과 같다.)
