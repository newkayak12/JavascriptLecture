EVENT

EVENT 속성 알아보기

input태그 관련 이벤트
focus : input태그에 focus되었을 때 발생 * 주의할 것은 focus가 되면서 blur가 같이 처리됨(blur랑 focus랑 같이 처리되면 무한 루프를 돈다.)
blur : input태그에서 focus가 사라졌을 때 발생
change : input에 있는 value속성의 값이 변경 되었을 때 발생 * select태그 이벤트에 많이 사용
submit : form 태그에서 submit타입을 클릭했을 때 발생(form태그로 데이터 전송 시 발생)
reset : form태그에서 reset타입을 클릭했을 때 발생
마우스 관련 이벤트
click : 마우스 좌클릭 했을 때 발생하는
dbClick : 마우스 더블 클릭 했을 때 발생하는
mouseEnter : 지정한 영역 안으로 마우스가 들어왔을 때 발생하는
mouseLeave : 지정한 영역 안에서 밖으로 나갔을 때 발생하는
key 관련 이벤트
keyUp : 키가 올라왔을 때 발생하는
keyDown : 키가 내려갔을 때 발생하는
keyPress : 키가 눌렸을 떄 발생하는
윈도우 창 관련 이벤트_윈도우 페이지
load : 페이지의 태그들이 다 로드됐을 때 발생하는 * 페이지 관련 이벤트
resize : 윈도우의 크기가 조정됐을때 발생하는
scroll : 창에서 스크롤이 조정됐을 때 발생하는
